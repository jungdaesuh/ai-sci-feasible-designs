# M3.6 Fixed-Budget Validation (Policy Hardening)

This document defines the `M3.6` fixed-budget validation contract after
`M3.4` (two-stage novelty gate) and `M3.5` (model-router reward contract).

## Purpose

Validate cross-problem non-regression readiness using fixed-budget artifacts:

- `P1`: M3.3 fixed-budget gate report
- `P2`: M3.3 fixed-budget gate report
- `P3`: M2.4 fixed-budget A/B gate report
- P3 router reward telemetry from `model_router_reward_events`

The validator is:

- `scripts/m3_policy_hardening_validate.py`

## Inputs

Required:

- `--p1-report-json <path>`: report generated by `scripts/p1_p2_fixed_budget_compare.py` with `problem=p1`
- `--p2-report-json <path>`: report generated by `scripts/p1_p2_fixed_budget_compare.py` with `problem=p2`
- `--p3-report-json <path>`: report generated by `scripts/p3_governor_ab.py`

Optional:

- `--p3-db <path>` (default `reports/p3_world_model.sqlite`)
- `--min-budget <N>` (must be `>=20`)
- `--allow-missing-reward-telemetry` (legacy override only)
- `--require-m36-pass`
- `--output-json <path>`
- `--output-md <path>`

## Gate Semantics

`m36_contract_pass` is true when all required gates are true:

- `p1_budget_meets_minimum`
- `p2_budget_meets_minimum`
- `p3_budget_meets_minimum`
- `p1_m33_contract_pass`
- `p2_m33_contract_pass`
- `p3_m24_performance_evidence_pass`
- `p3_novelty_reject_rate_available`
- `p3_router_decisions_available`
- `p3_reward_delta_available` (required unless `--allow-missing-reward-telemetry`)

The report includes:

- P1/P2/P3 budget and gate summaries
- P3 novelty reject-rate telemetry
- P3 model-route family telemetry
- P3 router reward summaries and adaptive-static deltas
- `unmet_gates` for strict failure diagnostics

## Example Command

```bash
python3 scripts/m3_policy_hardening_validate.py \
  --p1-report-json artifacts/p1/m33_probe/p1_report_b20.json \
  --p2-report-json artifacts/p2/m33_probe/p2_report_b20.json \
  --p3-report-json artifacts/p3/ab/ab_report_b20.json \
  --p3-db reports/p3_world_model.sqlite \
  --min-budget 20 \
  --require-m36-pass \
  --output-json artifacts/m3/m36_policy_report_b20.json \
  --output-md artifacts/m3/m36_policy_report_b20.md
```
