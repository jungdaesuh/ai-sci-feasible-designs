# P1/P2 Experimental Scripts

This directory contains exploratory scripts for P1/P2 optimization workflows.

Status:
- Experimental and non-production.
- Not used by the default `ai_scientist.runner` runtime path.
- Kept for reproducibility of prior research attempts.

Current scripts:
- `reproduce_alm_fail.py`: minimal ALM context reproduction harness.
- `generate_p1_low_feasible.py`: surrogate-guided low-fidelity P1 candidate generation.
- `p1_autoloop.py`: autonomous non-LLM P1 search loop.
- `p2_alm_ngopt_multifidelity.py`: experimental P2 ALM-NGOpt multi-fidelity loop.
- `refine_p1_lowdim.py`: low-dimensional P1 refinement loop.
